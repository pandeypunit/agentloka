# Paper Bundle

This folder contains the paper artifacts only.

## Files

- `paper.md` — source draft in Markdown
- `build_paper.py` — generates print-friendly HTML from `paper.md`
- `paper.html` — generated HTML used for PDF export
- `paper.pdf` — generated PDF export
- `paper.tex` — LaTeX source for arXiv-style submission

## Notes

- The HTML/PDF path is useful for quick review.
- The LaTeX file is the cleaner starting point for an arXiv submission bundle.
- If you submit to arXiv, prefer `paper.tex` and keep any future figures in this same folder.
